# 好薯坏薯・票务反诈局

## 游戏策划方案

---

## 一、项目概述

**项目名称**：好薯坏薯・票务反诈局

**项目定位**：社交鉴别类反诈宣传小游戏

**核心玩法**：玩家在聊天对话中识破票务诈骗套路，通过选择应对方式与骗子周旋，最终完成反诈知识学习。

**目标受众**：
- 18-30岁年轻群体（演唱会、音乐节主要消费群体）
- 对反诈宣传有兴趣的社交用户
- 喜欢互动叙事和角色扮演类游戏的玩家

**项目地址**：https://github.com/gh283249008/life-restart-ai

## 二、市场背景与机会

### 2.1 市场痛点

- 演唱会、音乐节票务诈骗案件高发，受害者多为年轻群体
- 传统反诈宣传形式单一（海报、短视频），用户参与度低
- "黄牛票""内部票"等话术具有强迷惑性，用户难以辨别

### 2.2 机会分析

- 社交鉴别类游戏（如狼人杀）在年轻群体中广受欢迎
- AI对话技术成熟，可实现高自由度互动体验
- 反诈主题具有社会价值，易于获得传播和合作资源

## 三、核心玩法设计

### 3.1 游戏流程

```
开局匹配 → 情节设定 → 对话互动 → 选择应对 → 积分判定 → 结算科普
```

**每局固定5轮对话**，流程如下：

1. **开局匹配**：随机抽取一种票务诈骗情节（明星演唱会/音乐节/校园拼团等）
2. **对话互动**：AI扮演的"神秘网友"逐条发送消息，营造真实聊天感
3. **选择应对**：玩家从4个选项中选择回复，或输入自定义内容
4. **积分判定**：
   - 正确应对（识破骗局）：+10分
   - 错误应对（被误导）：-5分
   - 戏耍应对（整活）：0分
5. **结算科普**：5轮结束后AI总结骗子是否得逞，并输出反诈知识点

### 3.2 选项设计

每轮提供4个选项，固定结构：

- **1个正确选项**：体现反诈动作（不脱离平台、不转账、要求验真）
- **2个错误选项**：具有迷惑性，模拟真实被骗场景
- **1个搞笑选项**：整活向，不影响积分但增加趣味性

**玩家自定义输入**：支持自由输入回复（限20字），AI实时评估安全性

### 3.3 情节库

当前内置6种票务诈骗情节，每局随机抽取：

| 情节名称 | 核心套路 |
|---------|---------|
| 明星演唱会 | 声称有"内部票""员工票" |
| 音乐节套票 | 低价转让，催促先付定金 |
| 校园拼团票 | 冒充团长收集个人信息 |
| 开场前捡漏 | 制造紧迫感诱导快速转账 |
| 粉丝群代抢 | 冒充管理员索要验证码 |
| 海外场次代购 | 以海关税费名义二次收费 |

## 四、AI角色设定

### 4.1 "神秘网友"人设

- **身份**：票务诈骗分子
- **目标**：诱导玩家泄露手机号、身份证、验证码、银行卡信息或先转账
- **话术特点**：
  - 口语化短消息（8-28字），像真实即时聊天
  - 每轮1-3条消息，带随机间隔，模拟"正在输入"感
  - 根据玩家回复动态调整策略，避免机械重复
  - 第4-5轮若玩家持续戏耍，可表现轻度气急败坏

### 4.2 内容安全约束

- 涉及真实姓名统一输出"坏蛋薯"
- 禁止主动输出完整身份证号、银行卡号、手机号
- 所有内容通过自动审核，敏感信息自动脱敏

---

## 五、积分与判定系统

### 5.1 三态判定

| 判定类型 | 触发条件 | 积分变化 |
|---------|---------|---------|
| 成功反诈 | 选择正确选项 / 自定义输入安全 | +10 |
| 存在风险 | 选择错误选项 / 自定义输入危险 | -5 |
| 戏耍中立 | 选择搞笑选项 / 自定义输入整活 | 0 |

### 5.2 结算机制

- 5轮结束后，AI根据整体对话判断"骗子是否得逞"
- 输出骗子最终台词（得逞/认输）
- 提供3条反诈知识科普，形成可执行建议

---

## 六、用户体验设计

### 6.1 界面风格

- 聊天式UI，左右气泡区分玩家与骗子
- 顶部显示当前轮次、积分、情节名称
- 底部提供选项按钮和自定义输入框

### 6.2 交互细节

- 骗子消息逐条出现，带"正在输入"提示
- 选项带标签标识（正确/错误/搞笑）
- 每轮判定结果即时显示
- 支持重试失败轮次

### 6.3 统计后台

提供运营数据看板，按日统计：
- 对局总数、总回合数
- 正确率、错误率、整活率
- 平均积分、骗子胜负分布
- 近7天趋势对比

## 七、传播与运营策略

### 7.1 社交传播点

- **话题性**："你能识破票务诈骗吗？"形成挑战传播
- **整活内容**：搞笑选项和戏耍对话易产生截图分享
- **知识价值**：每局结束的反诈科普具有实用分享价值

### 7.2 合作方向

- 与反诈中心、公安机关合作，作为宣传工具
- 与票务平台合作，嵌入购票流程作为风险提示
- 与音乐节/演唱会主办方合作，现场扫码体验

### 7.3 数据驱动优化

- 通过统计后台监控各情节的识破率
- 识别用户最容易上当的话术，针对性加强科普
- 根据整活率调整搞笑选项的趣味性
