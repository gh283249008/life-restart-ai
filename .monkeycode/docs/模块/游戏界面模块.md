## 3. 关键函数

### 3.1 `loadRoundPack()`

- 保证主题存在。
- 调用 `generateRoundPack(...)`。
- 更新来源标签与错误状态。
- 逐条投递骗子消息并设置本轮选项。

### 3.2 `deliverScammerMessages(messages, token)`

- 用随机延迟模拟“对方输入中”。
- 通过 `deliveryToken` 防止旧任务污染新局。

### 3.3 `pickOption(optionId)`

- 写入用户消息到 `chatHistory`。
- 执行积分变更与文本判定。
- 记录统计并进入下一轮或结算。

### 3.4 `sendCustomReply()`

- 读取输入并截断到 20 字符。
- 调用 `evaluateCustomReply(...)`。
- 根据 verdict 更新积分。
- 记录统计并推进流程。

### 3.5 `endOrNextRound()`

- 第 5 轮时生成最终报告并结束。
- 其他轮次递增并继续加载。

### 3.6 `restartGame()`

- 清理局内状态。
- 重置检索会话。
- 新建 session 并开始第 1 轮。

## 4. UI 结构

- 顶部信息卡：轮次、积分、情节、来源。
- 聊天区：左右气泡区分玩家与骗子。
- 交互区：
  - 选项按钮区
  - 自定义输入区
  - 错误卡片与重试按钮
- 结算区：结果、总结、科普、重开。
