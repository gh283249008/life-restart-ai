# 核心服务模块

## 1. 文件位置

- `src/services/antiFraudGame.ts`

## 2. 主题与数据模型

### 2.1 主题池

`SCENARIO_THEMES` 包含 6 个主题：

- 明星演唱会
- 音乐节套票
- 校园拼团票
- 开场前捡漏
- 粉丝群代抢
- 海外场次代购

### 2.2 核心类型

- `ScenarioOption`
- `RoundPack`
- `RoundPackResult`
- `FinalReport`
- `RoundGenerationError`

## 3. 工具函数

### 3.1 安全处理

- `sanitizeUnsafeText(text)`
- `hasUnsafeText(text)`
- `enforceRoundPackSafety(pack)`

### 3.2 结构归一

- `normalizeRoundPack(raw)`
  - 选项补齐到 4 条
  - 分类固定为 1 correct / 2 wrong / 1 funny
  - 生成 `correctOptionId`

### 3.3 JSON 提取

- `extractJson(raw)`
  - 支持从 ```json 代码块提取
  - 支持从首个对象花括号提取

## 4. LLM 调用底层

### 4.1 `callLLM(messages, maxTokens, bypassCircuit, extraRetries)`

内置能力：

- 请求超时控制
- 按状态码重试
- 网络异常重试
- 全局熔断
- 请求并发复用

### 4.2 熔断参数

- `BREAKER_FAIL_THRESHOLD = 6`
- `BREAKER_COOLDOWN_MS = 8000`

## 5. 业务 API

### 5.1 `generateRoundPack(...)`

- 生成一轮骗子话术与应对选项。
- 带 AI 修复与检索兜底链路。
- 返回来源标记 `ai/rag`。

### 5.2 `evaluateCustomReply(...)`

- 对玩家自定义回复进行三态判定。

### 5.3 `generateFinalReport(...)`

- 基于整局上下文返回最终胜负和反诈建议。

## 6. 异常模型

`RoundGenerationError` 字段：

- `stage`
- `reason`
- `status`
- `rawContent`

页面据此给出可观测错误信息。
