# 统计模块

## 1. 文件位置

- 数据层：`src/services/statsStore.ts`
- 展示层：`src/views/StatsView.vue`

## 2. 数据结构

### 2.1 会话记录 `SessionRecord`

- `sessionId`
- `themeId/themeName`
- `startedAt/finishedAt`
- `finalScore`
- `finalResult`
- `roundChoices[]`

### 2.2 回合记录 `RoundChoiceRecord`

- `sessionId`
- `round`
- `choiceCategory`
- `isCorrect`
- `themeId/themeName`
- `timestamp`

### 2.3 聚合结果 `DayStats`

- 对局数、回合数
- 各类选择次数
- 平均积分
- 骗子得逞/认输次数

## 3. 存储策略

- 本地 key：`anti_fraud_stats_v1`
- 最多保留最近 500 条会话记录。

## 4. 服务方法

- `startSession(...)`
- `recordRoundChoice(...)`
- `finishSession(...)`
- `getDayStats(dateStr?)`
- `getRecentDayStats(days)`

## 5. 页面展示逻辑

`StatsView.vue` 展示：

- 日期选择（近 7 天）
- 今日核心指标卡片
- 选项分布进度条
- 骗子胜负统计
- 近 7 天趋势表格

## 6. 统计口径说明

- 正确率：`correctCount / totalRounds`
- 错误率：`wrongCount / totalRounds`
- 整活率：`funnyCount / totalRounds`
- 自定义率：`(custom_safe + custom_risky + custom_tease) / totalRounds`
- 平均积分：已完成会话 `finalScore` 均值，保留 1 位小数。
