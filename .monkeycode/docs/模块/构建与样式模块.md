# 构建与样式模块

## 1. 构建工具链

## 1.1 Vite

文件：`vite.config.ts`

配置要点：

- Vue 插件：`@vitejs/plugin-vue`
- 别名：`@ -> src`
- 开发端口：`3000`
- `allowedHosts`: `.monkeycode-ai.online`
- 代理：`/deepseek` 转发到 `https://api.deepseek.com`

## 1.2 TypeScript

文件：`tsconfig.json`

- 扩展 `@vue/tsconfig/tsconfig.dom.json`
- `include` 覆盖 `src/**/*` 与 `.vue`
- 路径别名与 Vite 保持一致

## 2. 脚本命令

来自 `package.json`：

- `npm run dev`：本地开发
- `npm run build`：类型检查 + 生产构建
- `npm run preview`：本地预览构建产物
- `npm run lint`：ESLint 自动修复
- `npm run format`：Prettier 格式化 `src/`

## 3. 样式体系

文件：`src/style.css`

### 3.1 Tailwind

- `@tailwind base/components/utilities`

### 3.2 基础样式

- `html` 字体：`PingFang SC`, `Microsoft YaHei`
- `body` 背景：浅色渐变

### 3.3 组件样式类

- `.game-card`
- `.stat-item`
- `.choice-button`

## 4. 产物结构

构建输出目录：`dist/`

- `dist/index.html`
- `dist/assets/*.js`
- `dist/assets/*.css`

## 5. 页面元信息

文件：`index.html`

- `<title>反诈游戏demo</title>`
- 挂载点：`<div id="app"></div>`
