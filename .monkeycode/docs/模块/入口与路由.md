# 入口与路由模块

## 1. 入口文件

文件：`src/main.ts`

职责：

- 创建 Vue 应用实例。
- 创建并挂载路由实例。
- 引入全局样式 `src/style.css`。

## 2. 路由定义

路由模式：`createWebHistory()`。

路由表：

- `/` -> `HomeView`
- `/game` -> `GameView`
- `/stats` -> `StatsView`

## 3. 根组件

文件：`src/App.vue`

职责：

- 提供统一导航栏与主容器布局。
- 使用 `router-link` 和 `$route.name` 实现当前菜单高亮。
- 渲染 `<router-view />` 作为页面出口。

## 4. 首页模块

文件：`src/views/HomeView.vue`

职责：

- 展示玩法说明与反诈重点。
- 提供“开始鉴别”入口跳转 `/game`。

## 5. 注意点

- 当前没有路由守卫。
- 当前没有 404 回退路由。
